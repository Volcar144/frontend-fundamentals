## What is this?

Frontend Fundamentals is a structured curriculum that takes you from zero to production-ready across four essential modern web technologies:

| Topic | What you'll learn |
|---|---|
| [TypeScript](/docs/typescript) | Static types, interfaces, generics, utility types |
| [Tailwind CSS](/docs/tailwind) | Utility-first styling, responsive design, dark mode |
| [UX/UI Design](/docs/ux-ui) | Visual hierarchy, accessibility, colour theory |
| [Next.js](/docs/nextjs) | App Router, Server Components, data fetching, API routes |

## Recommended order

1. **TypeScript** — Learn the type system before anything else. You'll use it everywhere.
2. **Tailwind CSS** — Style your components quickly and consistently.
3. **UX/UI Design** — Understand *why* good interfaces look and feel the way they do.
4. **Next.js** — Bring it all together in a full-stack React framework.
5. **[Projects](/docs/projects)** — Apply everything in five hands-on builds.

## Prerequisites

- Basic JavaScript knowledge (variables, functions, arrays, objects)
- A code editor (VS Code recommended)
- Node.js 18+ installed

## Quick start

```bash
# Clone the example repo
git clone https://github.com/Volcar144/frontend-fundamentals
cd frontend-fundamentals
npm install
npm run dev
```
